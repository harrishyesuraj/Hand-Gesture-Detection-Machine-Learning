# Image-Based Hand Gesture Recognition Using CNN

## Academic Project | Second Year

This module focuses on recognizing hand gestures from image data using
a Convolutional Neural Network (CNN).

---

## Description
Hand gesture images were collected and preprocessed to create a custom
dataset. A CNN model was designed and trained to classify different
hand gestures. The implementation and training were performed using
Google Colab.

---

## Technologies Used
- Python
- Machine Learning
- Convolutional Neural Networks (CNN)
- Google Colab

---

## Dataset
- Self-collected hand gesture image dataset
- Used for training and testing the CNN model
- Dataset details are provided in the `dataset/` folder

---

## Implementation
- Notebook: `gesture_recognition_cnn.ipynb`
- Includes data preprocessing, model training, and evaluation
